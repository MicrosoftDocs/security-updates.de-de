Microsoft Security Bulletin MS15-090 – Hoch
===========================================

Sicherheitsanfälligkeiten in Microsoft Windows kann Erhöhung von Berechtigungen ermöglichen (3060716)
-----------------------------------------------------------------------------------------------------

Veröffentlicht: 11. August 2015

**Version:** 1.0

Kurzzusammenfassung
-------------------

Dieses Sicherheitsupdate behebt Sicherheitsanfälligkeiten in Microsoft Windows. Die Sicherheitsanfälligkeit ermöglicht eine Erhöhung von Berechtigungen, wenn ein Angreifer sich bei einem System anmeldet und eine speziell gestaltete Anwendung ausführt oder einen Benutzer dazu verleitet, eine speziell gestaltete Datei zu öffnen, welche die anfällige Sandboxanwendung ausführt, sodass der Angreifer der Sandbox entkommen kann.

Dieses Sicherheitsupdate wird für alle unterstützten Versionen von Microsoft Windows mit Ausnahme von Windows 10, das nicht betroffen ist, als **Wichtig** eingestuft. Weitere Informationen finden Sie im Abschnitt **Betroffene Software**.

Das Sicherheitsupdate behebt die Sicherheitsanfälligkeiten, indem korrigiert wird, wie Windows-Objekt-Manager die von einem Sandboxprozess erzeugten symbolischen Objektverknüpfungen verarbeitet. Dazu werden nicht ordnungsgemäße Interaktionen zwischen Sandboxanwendungen und der Registrierung sowie zwischen Sandboxanwendungen und dem Dateisystem unterbunden. Weitere Informationen zu diesen Sicherheitsanfälligkeiten finden Sie im Abschnitt **Informationen zu Sicherheitsanfälligkeiten**.

Weitere Informationen zu diesem Update finden Sie im [Microsoft Knowledge Base-Artikel 3060716](https://support.microsoft.com/de-de/kb/3060716).

Betroffene Software
-------------------

Die folgenden Softwareversionen oder -Editionen sind betroffen. Versionen oder Editionen, die nicht aufgeführt sind, haben entweder das Ende ihres Supportlebenszyklus überschritten oder sind nicht betroffen. Besuchen Sie die Website [Microsoft Support Lifecycle](http://support.microsoft.com/lifecycle), um den Supportlebenszyklus für Ihre Softwareversion oder Edition zu ermitteln.

Informationen zu Sicherheitsanfälligkeiten
------------------------------------------

Sicherheitsanfälligkeit in Windows-Objekt-Manager bezüglich der Erhöhung von Berechtigungen – CVE-2015-2428
-----------------------------------------------------------------------------------------------------------

In Windows-Objekt-Manager liegt eine Sicherheitsanfälligkeit bezüglich der Erhöhung von Berechtigungen vor, wenn Identitätswechselebenen nicht ordnungsgemäß überprüft und erzwungen werden. Ein Angreifer, der diese Sicherheitsanfälligkeit erfolgreich ausnutzt, kann die Sicherheitsprüfungen auf Identitätswechselebene umgehen und auf dem Zielsystem höhere Berechtigungen erlangen.

Um diese Sicherheitsanfälligkeit ausnutzen zu können, muss sich ein Angreifer bei einem betroffenen System anmelden und eine speziell gestaltete Anwendung ausführen. Für sich genommen ermöglicht diese Sicherheitsanfälligkeit nicht das Ausführen beliebigen Codes. Allerdings kann ein Angreifer diese Sicherheitsanfälligkeit in Verbindung mit einer anderen Sicherheitsanfälligkeit nutzen, um die Ausführung von beliebigem Code zu ermöglichen. Das Update behebt die Sicherheitsanfälligkeit, indem korrigiert wird, wie Windows-Objekt-Manager die von einem Sandboxprozess erstellten symbolischen Objektverknüpfungen verarbeitet.

Microsoft hat durch eine koordinierte Offenlegung der Sicherheitsanfälligkeit Informationen zu dieser Sicherheitsanfälligkeit erhalten. Microsoft lagen zum Zeitpunkt der Erstveröffentlichung dieses Security Bulletins keine Informationen vor, dass diese Sicherheitsanfälligkeit für Angriffe auf Benutzer ausgenutzt wurde.

### Schadensbegrenzende Faktoren

Für diese Sicherheitsanfälligkeit gibt es noch keine [schadensbegrenzenden Faktoren](https://technet.microsoft.com/de-de/library/security/dn848375.aspx).

### Problemumgehungen

Für diese Sicherheitsanfälligkeit gibt es noch keine [Problemumgehungen](https://technet.microsoft.com/de-de/library/security/dn848375.aspx).

Sicherheitsanfälligkeit in Windows-Registrierung bezüglich der Erhöhung von Berechtigungen – CVE-2015-2429
----------------------------------------------------------------------------------------------------------

In Microsoft Windows liegt eine Sicherheitsanfälligkeit bezüglich der Erhöhung von Berechtigungen vor, wenn von anfälligen Sandboxanwendungen aus bestimmte nicht ordnungsgemäße Interaktionen mit der Registrierung zugelassen werden. Ein Angreifer, der diese Sicherheitsanfälligkeit erfolgreich ausgenutzt hat, kann missbräuchlich mit der Registrierung interagieren und versuchen, der Anwendungssandbox zu entkommen.

Um die Sicherheitsanfälligkeit auszunutzen, muss ein Angreifer den Benutzer dazu verleiten, eine speziell gestaltete Datei zu öffnen, welche die anfällige Sandboxanwendung ausführt, wodurch die Sandbox beschädigt werden kann. Der Angreifer kann dann Programme mit den Berechtigungen eines angemeldeten Benutzers ausführen. Das Sicherheitsupdate behebt die Sicherheitsanfälligkeit, indem nicht ordnungsgemäße Interaktionen zwischen Sandboxanwendungen und der Registrierung unterbunden werden.

Microsoft hat durch eine koordinierte Offenlegung der Sicherheitsanfälligkeit Informationen zu dieser Sicherheitsanfälligkeit erhalten. Microsoft lagen zum Zeitpunkt der Erstveröffentlichung dieses Security Bulletins keine Informationen vor, dass diese Sicherheitsanfälligkeit für Angriffe auf Benutzer ausgenutzt wurde.

### Schadensbegrenzende Faktoren

Für diese Sicherheitsanfälligkeit gibt es noch keine [schadensbegrenzenden Faktoren](https://technet.microsoft.com/de-de/library/security/dn848375.aspx).

### Problemumgehungen

Für diese Sicherheitsanfälligkeit gibt es noch keine [Problemumgehungen](https://technet.microsoft.com/de-de/library/security/dn848375.aspx).
