Verwalten der Protokollierung
=============================

Auf der Seite **Protokollierungseinstellungen** werden der Name des Protokollierungsservers und der Datenbank des aktuellen Servers oder Clusters angezeigt. Auf dieser Seite können Sie den Protokollierungsdienst aktivieren und deaktivieren sowie Informationen zur Protokollierungsdatenbank anzeigen.

Zum Verwalten der Protokollierungsdatenbank wird Microsoft® SQL Server™ Enterprise Manager verwendet. Sie können die Protokolldateien anzeigen, die Größe des Protokolls durch Filtern der protokollierten Informationen verwalten und die Protokollierungsdatenbank auf einen anderen Server verschieben.

Zum Anzeigen von Fehlermeldungen zum Protokollierungsdienst können Sie die Ereignisanzeige verwenden.
